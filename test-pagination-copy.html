📋 Testing Instructions

How to Test:

  1. Setup Test Data: คลิก "Setup Test Data" เพื่อเพิ่ม patterns ที่มีจำนวนมาก
  2. Test Pagination: ดูว่ามี pagination controls ปรากฏขึ้นสำหรับ patterns ที่มีมากกว่า 1,000 รายการ
  3. Test Copy: คลิกที่เลขบัญชีใดๆ เพื่อ copy และดู "Copied!" feedback
  4. Test CSV Export: คลิก "Export เป็น CSV" และตรวจสอบ column order และชื่อ
  5. Test Navigation: ใช้ pagination controls เพื่อเปลี่ยนหน้า

Expected Results:

  • ✅ แสดงผลหน้าละ 1,000 รายการ
  • ✅ มี pagination controls สำหรับ patterns ที่มีมากกว่า 1,000 รายการ
  • ✅ คลิกเลขบัญชีแล้ว copy ได้ (ไม่มี dash)
  • ✅ แสดง "Copied!" feedback เมื่อ copy สำเร็จ
  • ✅ CSV export มี column order: account_number, pattern, pattern_type
  • ✅ CSV filename เป็น "account-numbers-[timestamp].csv"
